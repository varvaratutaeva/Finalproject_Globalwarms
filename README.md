Сайт о глобальном потеплении с тг-ботом.

На сайте находится информация об истории, причинах и последствиях глобального потепления, а также советы для решения глобальной проблемы.
Тг-бот может ответить на некоторые вопросы о климатических изменениях

Чтобы проверить как работает данный сайт вам нужно:
1 скопировать код из файла index.html
2 сохранить на своём компьютере с расширением .html
3 открыть файл в любом современном браузере или просто дважды щёлкнуть по файлу после сохранения.
